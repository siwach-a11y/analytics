What is AgentHub?

AgentHub is a marketplace of AI-powered bots that help users find tickets, travel, loans, currency rates, car rentals, trending videos, and deals. Each agent includes search filters, AI chat assistance, and Ticket Sniper Bot integration to monitor prices and auto-book when targets are met.

Platform features

  • Trending Video pipeline — discover → queue → approve → caption → publish (Fando-inspired)
  • Ticket Sniper Bot — automated hunts with live status, queue position, CAPTCHA/OTP handoff
  • Snipe Mode — proactive route/budget hunts (Flight & Hotel Finder)
  • 2.5% platform fee — charged on every bot snipe
  • AI assistants — per-agent chat with travel, finance, and deal tips
  • Lemon green theme — glass UI, marketplace search, category filters

Monetization

AgentHub charges a 2.5% platform fee on every bot snipe, calculated from the estimated snipe value. Users accept the fee before starting a hunt.

Run locally

  • Dev server: npm run dev → http://localhost:3000
  • Marketplace: /
  • This summary: /summary.html
  • Static build: npm run build:html → out/

Static build runs snipe hunts in browser demo mode. AI chat requires a server API key.

Project paths

  • Project root: ~/Projects/agenthub
  • Agent data: lib/data/agents.ts
  • Video discovery: lib/video-discovery/engine.ts
  • Snipe UI: components/sniper/
