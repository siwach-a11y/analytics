AgentHub — Static HTML Demo
===========================

QUICK START (easiest)
---------------------
macOS: double-click OPEN-DEMO.command
       (starts a local server — recommended)

Or in Terminal, from this folder:
  npx serve .

Then open http://localhost:3000 in your browser.
Start at summary.html or index.html


ALSO WORKS OFFLINE
------------------
You can double-click index.html directly — styles and navigation
should load without a server (fixed for file:// viewing).


WHAT'S INCLUDED
---------------
- summary.html        Platform overview (start here for review)
- index.html          Marketplace home
- agents/             All 7 agent demos
- _next/              Styles & JavaScript (required — do not delete)
